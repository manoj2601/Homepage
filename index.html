Feb. 2020 - May 2020

K-Medians in a Directed Tree

Prof. Smruti Ranjan Sarangi
(CSE Department, IIT Delhi)

The project aims to find the optimal placement of cache proxies in a computer network maintaining minimum overall cost. We moduled a C++ program that computes k-medians in a directed tree (edges directed from child to parent) in O(Pk2).
It is implemented in 2 phases and each phase uses a space of O(nk) complexity that is better than already established O(n2k) space complexity. This method employs dynamic programming for efficiency whereas best known algorithm for undirected tree is NP-hard.

  • 'k' is the number of resources to be placed, 'P' is path length of tree and 'n' is the number of vertices in the tree

August 2020 - Prsent

Incrypted Voting Protocol

Prof. Subodh Sharma
(CSE Department, IIT Delhi)

Semester long project that aims to an individually verifiable voting protocol with complete recorded-as-intended and counted-as-recorded guarantees. Protocol does rely on several cryptographic constructs to establish its correctness and maintain indiv. & community vote secrecy. It uses random distribution of pre-printed ballots with optional cast-or-audit component and optionally supports voter verifiable paper audit trails(VVPAT).
